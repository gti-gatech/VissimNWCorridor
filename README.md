# Northwest Corridor Vissim Model
This repository contains a PTV Vissim microscopic traffic simulation model developed to analyze driving behavior under managed lane toll pricing using dynamic traffic assignment (DTA).
- The VISSIM model requires the vissim model(.inpx file) and all 162 traffic signal files(.prbc) to be located together in the same directory.

## Requirements
- PTV Vissim 2023
- DTA add ons
